The Sticky Note Inspired CSS Theme

Stickynote is a CSS Theme inspired by the simple yet useful design of Sticky Notes. It's also designed to be as lightweight and as simple as possible. This theme was designed to be simple and delightful to use and very easy to integrate. One note to those who integrate it, it uses Google Fonts for its fonts. The fonts used in this theme are Indie Flower by Kimberly Geswein and Nerko One by Nermin Kahrimanovic. These fonts are licensed using the Open Font License (OFL).

Text Classes

Regular Text - Indie Flower - 18px Size

H4 - Nerko One - 22px Size

H3 - Nerko One - 28px Size

H2 - Nerko One - 32px Size

H1 - Nerko One - 48px Size

Footer - Indie Flower - 12px - 200 Weight
